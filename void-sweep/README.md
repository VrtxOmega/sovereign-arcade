## Overview

Void Sweep is a high-fidelity asteroid defense game built under the **VERITAS Omega** design system. Navigate hostile debris fields, destroy rogue asteroids, and upgrade your vessel through escalating sweep layers.

Built as a single-file application — zero dependencies, zero build step, pure HTML5 Canvas + JavaScript.

## Features

- **Gold-and-Obsidian Design** — Premium VERITAS aesthetic with wireframe asteroids, particle explosions, and screen shake.
- **Classic Asteroids Mechanics** — Thrust physics, screen wrapping, asteroid splitting (large → medium → small).
- **Upgrade Terminal** — Spend score on Pulse Frequency, Bolt Velocity, Thrust Output, Deflector Shields, and Hull Repair.
- **Persistent Progression** — Career profile tracks lifetime kills, layers cleared, score earned, and clean sweeps.
- **12 Trophies** — Unlock achievements from "First Contact" to "Void Legend" (Layer 20).
- **Leaderboard** — Top 10 high scores sorted by layer reached.
- **Deflector Shield** — Purchasable temporary invulnerability bubble (Shift to deploy).
- **LocalStorage Saves** — Career data persists across sessions.

## Controls

| Input | Action |
|-------|--------|
| **W / ↑** | Thrust forward |
| **A / ←** | Rotate left |
| **D / →** | Rotate right |
| **Space / X** | Fire |
| **Shift** | Deploy Deflector Shield |
| **P / Escape** | Pause |
